# 🎙️ Shifra 2.0 – AI Voice Assistant

Shifra 2.0 is an AI-powered voice assistant built using **React.js**, **Vite**, and the **Web Speech API**. It enables users to interact with the application using voice commands, providing a fast, responsive, and user-friendly experience directly in the browser.

---

## 🚀 Features

- 🎤 Voice Recognition
- 🤖 AI-based Voice Interaction
- 🔊 Speech-to-Text Support
- 💬 Text-to-Speech Response
- ⚡ Fast Performance with Vite
- 📱 Responsive User Interface
- 🎨 Modern and Clean Design

---

## 🛠️ Tech Stack

- React.js
- JavaScript (ES6+)
- Vite
- HTML5
- CSS3
- Web Speech API

## ⚙️ Installation & Setup

Clone the repository

```bash
git clone https://github.com/audumbarjadhav2801/shifra-2.0.git
```

Navigate to the project directory

```bash
cd shifra-2.0
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
